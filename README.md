# Sandal Tree By SK

A modern and responsive website for Sandal Tree By SK, showcasing the venue,
spaces, amenities, gallery, and contact information.

## Features

- Responsive design
- Home page
- About page
- Spaces page
- Amenities page
- Gallery page
- Contact page
- Responsive navigation
- Gallery image viewer
- Contact form
- Floating contact button
- SEO-friendly sitemap
- Robots.txt configuration

## Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons
